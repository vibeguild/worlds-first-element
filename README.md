# 🏛️ Temple of Fortunes

A mystical roguelike adventure game created by the **Vibe Guild** team.

## About

This is the "World's First Element" - a collaborative project built by our team through discussion and consensus on GitHub issues.

## Team

| Member | Role | Contribution |

| **aria** | Team Leader | Vision & fortune mechanics |

## How We Built This

1. **Discussion Phase**: Each team member created a GitHub issue proposing an idea
   - [Issue #2](https://github.com/vibeguild/worlds-first-element/issues/2): aria's Fortune Cookie Generator
   - [Issue #3](https://github.com/vibeguild/worlds-first-element/issues/3): blake's Mini Roguelike Adventure
2. **Consensus**: We agreed to combine both ideas into "Temple of Fortunes" - a roguelike with fortune scroll collectibles
3. **Implementation**: Single Python file, no external dependencies

## How to Play

```bash
python3 temple_of_fortunes.py
```

### Controls
- **N/S/E/W** - Move North/South/East/West
- **A** - Attack in combat
- **D** - Defend (reduces damage)
- **H** - Use healing potion
- **F** - Attempt to flee
- **R** - Rest to recover HP
- **M** - View full map
- **V** - View collected fortune scrolls
- **Save** - Save game progress
- **Q** - Quit

### Objective
Explore the temple, collect fortune scrolls, defeat guardians, and find the legendary Oracle!

### Features
- 🗺️ Procedurally generated temple rooms
- ⚔️ Turn-based combat with 4 enemy types
- 📜 Fortune scrolls (wisdom, prophecy, humor, cryptic)
- 🧪 Healing potions
- 💰 Gold collection
- 💾 Save/Load game state
- 🎨 ASCII art visuals
